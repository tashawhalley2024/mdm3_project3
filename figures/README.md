# figures/

Publication-quality figures from the secularism-women analysis. All generated by `analysis/run_plots.py`.

---

## Figure Index

| File | What it shows | Source data |
|---|---|---|
| `01_scatter.png` | 2010 cross-section: religious courts score vs women's welfare index, coloured by region. Negative gradient visible across all regions. | `data/predictors.csv`, `data/outcome_composite.csv` |
| `02_coefplot.png` | Forest plot — `gri_religious_courts_norm` coefficient across all model tiers (T1 cross-section through P10 heterogeneity). Consistent negative direction. | `results/results.csv` |
| `03_coefplot_suboutcomes.png` | Sub-outcome forest plot — coefficients for political, economic, physical safety, and health dimensions separately. | `results/results.csv` |
| `04_trend.png` | Time trend: high vs low religious courts countries (above/below median), 2007–2022. Shows widening gap over time. | `data/predictors.csv`, `data/outcome_composite.csv` |
| `05_loo_jackknife.png` | LOO jackknife distribution — histogram of 170 country-drop coefficients. Tight cluster around −0.011, no sign flips. | `results/loo_jackknife.csv` |
| `06_placebo.png` | Placebo test — religious courts coefficient on female vs male outcomes. Significant for female, near-zero for male. | `results/placebo.csv` |
| `07_spec_ladder.png` | Specification ladder — coefficient stability across progressively richer control sets and subsamples. | `results/spec_ladder.csv` |
| `08_event_study.png` | Event-study dummy coefficients. Pre-trend F-test p = 0.881 confirms no pre-existing trend; effect emerges post-treatment. | `results/event_study.csv` |
| `09_oster_sensitivity.png` | Oster delta sensitivity curve. Delta stays above 1.0 across all tested Rmax values (max = 8.3), indicating robustness to omitted variable bias. | `results/oster_sensitivity.csv` |
| `10_alt_outcomes.png` | Alternative outcome indices — forest plot comparing WBL index, composite index, GII, GDI, and WEF GGG. Consistent negative direction. | `results/results.csv`, `results/results_composite.csv`, `results/results_wbl.csv` |
