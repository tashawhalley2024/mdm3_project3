Publication-quality figures from the secularism-women analysis. All generated by `analysis/run_plots.py`.

## Figure Index

| File | What it shows | Source data |
|---|---|---|
| `00_map.png` | World choropleth — religious courts institutionalisation by country (2020, Pew GRI). Anchor country labels so the colour scale reads concretely. | `data/predictors.csv` |
| `01_scatter.png` | Dual-panel 2020 scatter: courts (null, left) vs apostasy (strong negative slope, right), each with OLS fit + 95% CI and country anchors. | `data/predictors.csv`, `data/outcome_wbl.csv` |
| `02_coefplot.png` | Forest plot across T1 (2020 cross-section) and T2 (panel FE). Apostasy + courts highlighted at top; context predictors greyed below. | `results/results.csv` |
| `05_loo_jackknife.png` | Leave-one-out jackknife, 2x2 grid: histogram of country-drop coefficients (top) and 5 most influential countries (bottom), for courts (left) and apostasy (right). | `results/loo_jackknife.csv`, `results/loo_jackknife_apostasy.csv` |
| `06_placebo.png` | Gendered-mechanism test: apostasy coefficient on female DVs (foregrounded) vs male DVs (desaturated). Women's treatment index leads. | `results/placebo_apostasy.csv` |
| `07_spec_ladder.png` | Dual-panel specification ladder: Oster controls ladder stacked on sensitivity/subsample checks for courts (left) vs apostasy (right). | `results/spec_ladder.csv`, `results/spec_ladder_apostasy.csv`, `results/results.csv` |
| `09_oster_sensitivity.png` | Oster δ sensitivity curve (apostasy). Robust region shaded; headline δ at Rmax = 2.2× annotated. | `results/oster_sensitivity_apostasy.csv` |
| `10_alt_outcomes.png` | Forest plot: courts coefficient across the primary composite and external gender indices (WEF GGG, UNDP GDI, WBL legal rights). | `results/results.csv`, `results/results_wbl.csv` |
| `11_wbl_groups.png` | Courts coefficient across the 10 WBL legal-rights domains plus overall composite (2013–2022). Sorted by coefficient; null across every domain. | `results/results_wbl_groups.csv` |
| `12_mundlak_decomposition.png` | Hero figure. Paired within-country (left) vs between-country (right) effects from T4 Mundlak RE-FE hybrid. Apostasy row highlighted; between/within ratio annotated. | `results/results.csv` (T4_mundlak_re rows) |

**Removed from the presentation set:**
- `03_coefplot_suboutcomes.png` — underlying Phase 3 sub-outcome tiers are no longer in `results.csv`.
- `04_trend.png` — redundant with `12_mundlak_decomposition.png`, which makes the structural vs temporal argument more rigorously.
- `08_event_study.png` — event studies are not informative for slow-moving institutional variables (73% of courts variation is between-country).
- Courts-version Oster sensitivity — applying Oster to a null predictor is logically inverted; only the apostasy version is retained.
