# figures/

Publication-quality figures from the secularism-women analysis. All generated by `analysis/plot_secularism_women.py`.

---

## Figure Index

| File | What it shows | Source data |
|---|---|---|
| `secularism_women_scatter.png` | 2010 cross-section: religious courts score vs women's welfare index, coloured by region. Negative gradient visible across all regions. | `data/secularism_composition_normalised.csv`, `data/women_secularism_normalised.csv` |
| `secularism_women_coefplot.png` | Forest plot — `gri_religious_courts_norm` coefficient across all model tiers (T1 cross-section through P10 heterogeneity). Consistent negative direction. | `results/secularism_women_results.csv` |
| `secularism_women_coefplot_suboutcomes.png` | Sub-outcome forest plot — coefficients for political, economic, physical safety, and health dimensions separately. | `results/secularism_women_results.csv` |
| `secularism_women_trend.png` | Time trend: high vs low religious courts countries (above/below median), 2007–2022. Shows widening gap over time. | `data/secularism_composition_normalised.csv`, `data/women_secularism_normalised.csv` |
| `secularism_women_loo.png` | LOO jackknife distribution — histogram of 170 country-drop coefficients. Tight cluster around −0.011, no sign flips. | `results/secularism_women_loo.csv` |
| `secularism_women_placebo.png` | Placebo test — religious courts coefficient on female vs male outcomes. Significant for female, near-zero for male. | `results/secularism_women_placebo.csv` |
| `secularism_women_spec_stability.png` | Specification ladder — coefficient stability across progressively richer control sets and subsamples. | `results/secularism_women_spec_stability.csv` |
| `secularism_women_event_study.png` | Event-study dummy coefficients. Pre-trend F-test p = 0.881 confirms no pre-existing trend; effect emerges post-treatment. | `results/secularism_women_event_study.csv` |
| `secularism_women_oster_sensitivity.png` | Oster delta sensitivity curve. Delta stays above 1.0 across all tested Rmax values (max = 8.3), indicating robustness to omitted variable bias. | `results/secularism_women_oster_sensitivity.csv` |
| `secularism_women_alt_outcomes.png` | Alternative outcome indices — forest plot comparing WBL index, composite index, GII, GDI, and WEF GGG. Consistent negative direction. | `results/secularism_women_results.csv`, `results/secularism_women_results_composite.csv`, `results/secularism_women_results_wbl.csv` |
