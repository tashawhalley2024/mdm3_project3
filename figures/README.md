The focal predictor `FOCAL_PRED = composite_secularism_norm` (Item 2, 2026-04-15) is an equal-weight z-score composite over six Pew GRI sub-items, V-Dem `v2clrelig` (sign-flipped), and four WVS religious-intensity items. Built in-memory at load time by `analysis/utils.py:build_secularism_composite`; not stored on disk. `FOCAL_PRED_2 = gri_apostasy_norm` is the secondary sub-item focal (strongest individual GRI component). `FOCAL_PRED_LEGACY = gri_religious_courts_norm` is preserved as a robustness focal with outputs suffixed `_religious_courts`; it does not drive the figure set.

**Removed from the presentation set:**
- `03_coefplot_suboutcomes.png` — underlying Phase 3 sub-outcome tiers are no longer in `results.csv`.
- `04_trend.png` — redundant with `12_mundlak_decomposition.png`, which makes the structural vs temporal argument more rigorously.
- `08_event_study.png` — event studies are not informative for slow-moving institutional variables.
- Oster sensitivity for the composite and legacy courts focals — applying Oster to a broad composite with multiple dimensions or to a known null sub-item is not informative; only the apostasy version is retained as the Oster headline.

## Composite-variant outputs (not plotted)

Follow-up #1, #2, and #3 add four robustness composite variants (`composite_secularism_real_norm`, `composite_secularism_instonly_norm`, `composite_secularism_covwt_norm`, `composite_secularism_pca_norm`). These surface in `results/headline_table.md` and in the methodology paper (`writing/methodology_full.tex` §2.2 and §6.2) but are not added to the figure deck — the 12-figure presentation set is frozen on the equal-weight composite headline to keep the narrative stable.
